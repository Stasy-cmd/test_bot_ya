## Тестовое задание 
*Выполнено для вакансии ["Наставник на курсы для подростков по Python-разработке" ](https://practicum.yandex.ru/job/vacancy-130)*

***Данный чат бот немного расскажет обо мне.***

#### Используемые технологии:
- aiogram
- emoji

#### Как запустить бота:
1. Создать своего бота в телеграмме
2. Скопировать ключ
3. Скачать данный репозиторий
4. Создать окружение
5. Установить библиотеки из "requirements.txt"
6. Вставить скопированный ключ в "config/settings.py" в переменную "bot_token"
7. Запустить "bot/menu_bot.py"
