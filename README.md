#### Как запустить бота:
1. Создать своего бота в телеграмме
(Пишем @BotFather и следуем инструкции)

2. Получить ключ у BotFather и сохранить его к себе в заметки 

3. Клонировать репозиторий и перейти в него в командной строке:
```
https://github.com/Mizantropus/api_yamdb.git](https://github.com/Stasy-cmd/test_ya.git
```

```
cd test_bot_ya
```

4. Cоздать и активировать виртуальное окружение:
```
python3 -m venv env
```

```
. env/bin/activate
```

5. Вставить сохраненный на 2 этапе ключ в "config/settings.py" в переменную "bot_token"
6. Установить зависимости из файла requirements.txt:
```
python3 -m pip install --upgrade pip
```

```
pip install -r requirements.txt
```

7. Запустить проект:
```
python3 main.py
